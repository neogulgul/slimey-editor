How To Use:
	B                              - Brush
	F                              - Fill
	E                              - Toggle Erase
	R                              - Reset View
	Control + Shift + R            - Clear Map
	Left Click (on Map)            - Action, Place / Erase (with either Brush or Fill)
	Left Click (on Tile Selection) - Select Tile
	Alt + Left Click               - Select Tile From Map (Eyedropper Tool)
	Middle Click + Drag            - Move Map
	MouseWheel                     - Zoom
	Control + S                    - Save Map (to map.txt)
